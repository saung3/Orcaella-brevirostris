## Dataset Summary
This dataset compromises of cetaceans MHCIIDQB Alleles from NIH-GenBank with Species annotation and GenBank Ascension numbers. The data set includes 223 sequences of MHCIIDQB Alleles downloaded from Genbank and Four MHCIIDQB Alleles of Irrawaddy dolphins. All the MHCIIDQB Alleles were trimmed to display only the exon 2 (121bp) of the MHC II DQB locus. 

## Data Format 
All the data are DNA sequences in standard FASTA files. The Ascension numbers of GenBank are displayed first along with using the first initial of the genus and three starting letters of the species. 

## Curation Rationale
This dataset was created to combine all the MHCIIDQB Alleles in one standard FASTA file for easier application on IQ tree tool of CIPRES. There has been a lack of information to determine the evolutionary aspect of the MHCIIDQB gene among cetaceans. Past studies have hypothesized the maintenance of functional MHC genes across cetaceans. Comparing MHCIIDQB gene with all the other cetaceans using a phylogeny will give a visual model on how DQB alleles evolve and diverge over time.
